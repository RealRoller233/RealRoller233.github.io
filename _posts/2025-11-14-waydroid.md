## 安装 Waydroid

- 使用 [waydroid_script](https://github.com/casualsnek/waydroid_script) 安装。
- **建议安装 Android 11**，Android 13 目前仍有问题，ARM 转译不可用。
- 可选组件：
  - `gapps`（可选）
  - `magisk`（可选）
- **必选组件**：
  - `libhoudini`
- 自定义 Waydroid 镜像：
  - [官方文档](https://docs.waydro.id/faq/using-custom-waydroid-images)
  - 镜像下载网站：[SourceForge Waydroid Images](https://sourceforge.net/projects/waydroid/files/images/)

## 使用技巧

- 文件传输：
  - 使用 `adb push` / `adb pull` 在主机与 Waydroid 之间传输文件。
- 安装 APK：
  - 使用 `adb install <apk_file>` 安装应用。
- 虚假触控模拟：
  ```bash
  waydroid prop set persist.waydroid.fake_touch <package_name>
